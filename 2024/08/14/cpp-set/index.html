C++: std::set

What’s set?

sets are associative containers that store unique elements and sorted in a certain order.

Set in C++

std::set provided efficient operations to search, insert and delete elements. It’s implemented based on a Red-Black Tree and therefore has find, insert and delete performance with logarithmic time complexity.

std::set is the class template for map containers and it’s defined inside the <set> header file.

elements which can be stored in std::set must follow these condition:

  • the type of elements must can be comparable;
  • the type of elements must can be removed and assigned;

Common Operations

method description
insert(elem) insert a element
erase(elem) remove a element
find(elem) search a element
size() return the number of element in set
empty check if the set is empty
