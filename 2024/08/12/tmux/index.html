Preface

Tmux is a terminal multiplexer. It lets you switch easily between several programs in one terminal, detach them (they keep running in the background) and reattach them to a different terminal.

Tmux Core Feature

  1. tmux can create multiple panes within a single window.

  2. after restarting the tmux, you can recover the last session through the commond line.

  3. there are lots of plugins that you can extend your tmux.

‚ÄºÔ∏è In tmux, all shortcuts must be preceded by the prefix key ctrl + b.

Pane Operations

  • % Split the window into two panes side by side.
  • " Split the window into two panes top and bottom.
  • x Close the current pane.
  • { Move the current pane one position forward.
  • } Move the current pane one position backward.
  • ; Select the last used panes
  • o Selected the next pane, or you can use the arrow key to select.
  • space Switch panel layout, tmux has 5 built-in layouts.
  • z Maximize the current pane, and execute again to restore the original size.
  • q Display the numbers of all panes, and press the corresponding while the number are displayed to switch to the corresponding pane.

Session Operations

In shell, each time you enter tmux, a tmux session is created.
Common tmux operations include:

  • $ Rename current session.
  • s Switch session list.
  • d Detach current session(You will exit the tmux process and return to the shell main process).
